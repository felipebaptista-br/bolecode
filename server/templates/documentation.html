BOLECODE API

Bolecode
Dynamic API

Essa é uma API dinâmica para os clientes do itaú. O principal objetivo é realizar requisições utilizando o Itaú for developers para realizar a geração de Boletos com QRcode

DOCUMENTAÇÃO

Documentação

/bolecode - Método POST que recebe como parâmetro um arquivo JSON com todos os dados necessários para a requisição e retorna um redirecionamento de página para a página do Boleto com opção de baixar.

/bolecode/run-test-json - Método GET que não contém nenhum parâmetro, esse endpoint serve para testar as requisições que o servidor faz com dados mocados e retorna um JSON com as respostas da API do Itaú.

/bolecode/run-test-boleto - Método GET que não contém nenhum parâmetro, esse endpoint serve para testar as requisições que o servidor faz com dados mocados e retorna um redirecionamento de página para a página do Boleto com opção de baixar.

/bolecode/acesstoken - Método GET, utiliza a certificação que está no Back-end para geração de um AcessToken, retorna o AcessToken em formato JSON.

Servidor Flask & Python

O motivo de escolhermos essa stack está na vantagem de bibliotecas que a linguagem Python possui e a performance de se utilizar um Microframework como o Flask para desenvolver simples aplicações
